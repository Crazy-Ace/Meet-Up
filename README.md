# Meet-up
Meet-Up Event, First Project for the Udacity Senior Web Developer Nanodegree

##Running
- Installing dependencies:
```{r, engine='bash', count_lines}
$ npm install
```

- Download packages:  
```{r, engine='bash', count_lines}
$ bower install
```

- Running:  
```{r, engine='bash', count_lines}
$ gulp serve
```
